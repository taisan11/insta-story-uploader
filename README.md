jpgまたはmp4がアップロード可能です。必ず16:9のサイズにしてください。

### Link Sticker の座標指定について
- X, Y, 幅, 高さはすべて 0〜1 の正規化値です（ストーリーキャンバス全体に対する割合）。
- X, Y はスタンプの中心座標です。0 が左/上、1 が右/下。
- 幅, 高さはスタンプのサイズをキャンバス比で指定します。0.5 なら横幅はキャンバスの 50% です。
- デフォルト値（UI にプリセット済み）:
	- X: 0.5126011
	- Y: 0.5168225
	- 幅: 0.50998676
	- 高さ: 0.25875
